# คู่มือใช้งาน ระบบแจ้งเตือนงานผ่าน LINE
## DaiClassroom Assignment Reminder System

---

## ✅ สำหรับนักเรียน

### 1️⃣ เชื่อมต่อบัญชี LINE กับระบบ

#### ขั้นตอน:
1. **เพิ่ม Official Account (OA)**
   - เปิด LINE
   - ค้นหา ID: `@676lsfsx` หรือชื่อห้องเรียนของคุณ
   - แตะปุ่ม **"เพิ่มเพื่อน"** (Add friend)

2. **ส่งรหัสนักเรียน**
   - เมื่อบอทเชื่อสวัสดี จะถามให้ส่งรหัสนักเรียน
   - พิมพ์เพียงตัวเลข เช่น `64001`
   - แตะส่ง

3. **ยืนยันการเชื่อมต่อ**
   - บอทจะตอบกลับ:
     ```
     เชื่อมต่อสำเร็จ! ✅
     
     ยินดีต้อนรับ [ชื่อของคุณ] 🎉
     รหัสนักเรียน: 64001
     
     คุณจะได้รับการแจ้งเตือนเมื่อมีงานใกล้ครบกำหนดส่ง
     ```
   - สำเร็จ! ✅ ตั้งแต่นี้ไปจะได้รับการแจ้งเตือนผ่าน LINE

#### ⚠️ หากเจอปัญหา:
- **"ไม่พบรหัสนักเรียน"** → ตรวจสอบรหัสให้ถูกต้อง หรือติดต่ออาจารย์
- **ส่งหลายครั้งแล้วยังไม่สำเร็จ** → ปิด LINE แล้วเปิดใหม่ หรือติดต่อ IT

### 2️⃣ รับการแจ้งเตือนงาน

#### เมื่อมีงานใกล้ครบกำหนด
- **เวลา**: ทุกวันเวลา **08:00 น.** (เช้า)
- **รูป**: ข้อความ LINE แสดง:
  ```
  สวัสดี [ชื่อคุณ] 👋
  
  แจ้งเตือน: คุณยังไม่ได้ส่งงาน
  📝 "[ชื่องาน]"
  
  📅 กำหนดส่ง: XX/XX/XXXX
  
  กรุณาส่งงานให้ทันกำหนดเพื่อไม่ให้เสียคะแนน 📚
  ```

#### ✨ ส่งงานได้ทันที
- ดำเนินการ (คลิก / กรอกข้อมูล) ในหน้า **ระบบจัดการหรือที่อาจารย์บอก**
- บอทจะแจ้ง "ส่งแล้ว ✅" โดยอัตโนมัติ

## 👨‍🏫 สำหรับครู

### 1️⃣ สร้างงานใหม่

#### ขั้นตอน:
1. **เปิดระบบ** → ไปที่ **"งานที่มอบหมาย"** (Assignments)

2. **คลิก "สร้างงานใหม่"** (ปุ่มสีส้ม)

3. **กรอกข้อมูลงาน**
   - **ชื่องาน**: เช่น "รายงานบทที่ 3" หรือ "ทดสอบออนไลน์"
   - **คำอธิบาย**: รายละเอียดเพิ่มเติม (optional)
   - **รายวิชา**: เลือกวิชาที่มีนักเรียนลงนาม
   - **กำหนดส่ง**: เลือกวันที่ (เบาะแสว่าเมื่อไหร่ต้องเสร็จ)

4. **คลิก "สร้างงาน"**
   - ระบบจะสร้าง **submission record** โดยอัตโนมัติ
   - สถานะเริ่มต้น: **"ยังไม่ส่ง"** สำหรับนักเรียนทั้งหมด

#### ✅ พร้อมแล้ว!
- งานจะปรากฏในรายการ
- นักเรียนจะได้รับแจ้งเตือนอัตโนมัติในเช้าของวันครบกำหนด

---

### 2️⃣ ติดตามการส่งงาน

#### ขั้นตอน:
1. **เข้าหน้า "งานที่มอบหมาย"**

2. **เลือกงานที่ต้องการตรวจสอบ**
   - คลิกเรื่องชื่องาน

3. **ดูสถานะส่ง**
   ```
   ส่งแล้ว 15/20 คน
   
   ✅ นักเรียนที่ส่อแล้ว:
   • นาย/น.ส. [ชื่อ] - ส่งแล้ว (ปุ่มสีเขียว)
   
   ❌ ยังไม่ส่อ:
   • นาย/น.ส. [ชื่อ] - ยังไม่ส่ง (ปุ่มสีเทา)
   ```

4. **ทำเครื่องหมายการส่ง** (ถ้าต้องการระบุด้วยมือ)
   - คลิกปุ่มแต่ละคน
   - เปลี่ยนสถานะเป็น **"ส่งแล้ว"** หรือ **"ยังไม่ส่ง"**
   - การตัดสินใจจะบันทึกอัตโนมัติ

#### 📊 เก็บสถิติ
- ระบบจดบันทึก **ใครส่ออย่างไร และเมื่อไหร่**
- สามารถดู progress bar เพื่อรู้เปอร์เซ็นต์นักเรียนที่ส่อแล้ว

---

### 3️⃣ ลบหรือแก้ไขงาน

#### แก้ไข:
1. คลิกปุ่ม **"แก้ไข"** ที่มุมขวา
2. ปรับชื่อ / คำอธิบาย / กำหนดส่ง
3. **"บันทึกการแก้ไข"**

#### ลบ:
1. คลิกปุ่ม **"ลบ"** (สีแดง)
2. ยืนยัน "ลบงาน"
3. งานและบันทึกการส่งจะลบถาวร

---

## 🔔 ระบบแจ้งเตือนใช้งานอย่างไร

### ตลาด 08:00 น. ทุกวัน
- **หางาน** ที่:
  - **เลยกำหนด** (แดง ⚠️)
  - **ครบกำหนดวันนี้** (สีส้ม ⏰)
  - **ครบกำหนดพรุ่งนี้** (เหลือง 📅)

- **หานักเรียน** ที่:
  - สถานะ = "ยังไม่ส่ง"

- **ส่ง LINE Push message** ไปยังคนนั้น:
  ```
  สวัสดี [ชื่อ] 👋
  
  แจ้งเตือน: คุณยังไม่ได้ส่งงาน
  📝 "[ชื่องาน]"
  
  ⚠️ งานนี้เลยกำหนดส่งไปแล้ว กรุณาติดต่ออาจารย์โดยด่วน
  หรือ
  📅 กำหนดส่ง: XX/XX/XXXX (พรุ่งนี้)
  
  กรุณาส่งงานให้ทันกำหนด 📚
  ```

### จดหมายสำาหรับครู
- ระบบบันทึก **log ทั้งหมด** ใน Firestore
- ดู logs: Firebase Console → Cloud Functions → Logs

## ❌ Troubleshooting

### นักเรียน
| ปัญหา | เหตุผลและวิธีแก้ |
|---|---|
| ไม่ได้รับแจ้งเตือนใน LINE | • ตรวจว่า link account สำเร็จหรือไม่ (ส่งรหัสนักเรียนใหม่)<br>• บอท offline → รอ 5 นาที |
| "ไม่พบรหัส 64001" | • ตรวจความถูกต้องกับอาจารย์<br>• ห้ามมีช่องว่างหรือตัวอักษร |
| ส่องงานแล้วแต่บอทยังบอก "ยังไม่ส่ง" | • ระบบใช้เวลา 1-2 นาทีอัปเดต<br>• รีเฟรชหน้า |

### ครู
| ปัญหา | เหตุผล และวิธีแก้ |
|---|---|
| งานที่สร้างไม่ปรากฏ | • ตรวจว่า teacherId และ subjects ถูกต้อง<br>• รีเหลดหน้า |
| ส่งปุ่ม "ส่งแล้ว" แต่สถานะไม่เปลี่ยน | • ลองรีเหลดหน้าหลังจาก 2 วินาที<br>• ตรวจ browser console สำหรับ error |
| ไม่เห็น reminder log | • Firebase Console → Functions → Logs<br>• กรอง: `sendAssignmentReminders` |
